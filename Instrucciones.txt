git init : Inicia una carpeta vacia en mi proyecto, esta oculta

git status: consulta como se encuentra el proyecto en ese momento 

Carpetas vacias: git no realiza un seguimiento de las carpetas vacias

.gitkeep con ese comnado podemos realizar un seguimiento a la carpeta vacia

.gitignote va a ignorar carpetas o archivos

1- git add "nombre del archivo" || git add . (Para agregar todo)

2- git commit -m "mensaje que queremos enviar"

3- Git push origin master / main
